# Fleet Management System - User Guide

## Introduction
Welcome to the Fleet Management System. This guide will help you navigate the application and perform daily tasks.

## Getting Started
1.  **Login**: Access the application URL and enter your credentials.
2.  **Dashboard**: The main dashboard shows an overview of fleet status, pending work orders, and recent alerts.

## Key Features

### Assets
- **View Assets**: Click "Assets" in the sidebar to see a list of all vehicles and equipment.
- **Asset Details**: Click on an asset to view its history, specifications, and maintenance logs.
- **Update Mileage**: Regular mileage updates are crucial for maintenance scheduling.

### Work Orders
- **Create Work Order**: Use the "New Work Order" button to report an issue.
- **My Tasks**: Technicians can view assigned work orders under "Inbox" or "Work Orders".
- **Complete Work Order**:
    1.  Open the work order.
    2.  Complete the checklist items.
    3.  Add parts used and labor hours.
    4.  Click "Complete".

### Inspections
- **Daily Check**: Operators should perform a pre-start inspection daily.
- **Process**:
    1.  Go to "Inspections" > "New Inspection".
    2.  Select the asset.
    3.  Follow the step-by-step checklist.
    4.  Sign and submit.

### Inventory
- **Parts Request**: If you need a part, check the "Inventory" section.
- **Stock Levels**: Warehouse managers can update stock levels and track usage.

## Support
If you encounter any issues, please contact your System Administrator.
